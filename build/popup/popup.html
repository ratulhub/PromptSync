    ⌨️ Slash Commands
    
      
    Type these at the start of your message to control injection per-message.
    
      
    
        
    /no-memorySkip memory injection for this message
    
        
    /only-profileInject only your profile (no saved notes)
    
        
    /strictUltra-minimal context (name + pinned only)
    
        
    /explainAllow full explanation (disables short-answer mode)
    
        
    /detailFull context + detailed answer
    
        
    /codeCode mode — include explanations with code
    
        
    /resetRevert to default behavior
    
        
    /temp <text>Add session-only temporary memory
